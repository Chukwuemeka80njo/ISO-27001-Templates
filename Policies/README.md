# Policies

This folder contains the complete set of Information Security Management System (ISMS) policies required to support ISO/IEC 27001:2022 compliance.

Policies provide the governing principles, expectations, and requirements that guide how information security is managed across the organization.  
They form the foundation for procedures, controls, risk treatment, and operational security activities.
