# 01_ISMS-Scope

This folder contains all documentation related to defining the scope of the Information Security Management System (ISMS) according to ISO/IEC 27001:2022.

The ISMS scope establishes:
- The boundaries and applicability of the ISMS  
- Organizational units, processes, systems, and locations included  
- Interfaces and dependencies  
- Justification for any exclusions  

### Included Templates
- ISMS Scope Statement  
- ISMS Boundary & Applicability Matrix  
- ISMS Assets & Processes Covered  
- Assumptions, Dependencies, and Constraints
