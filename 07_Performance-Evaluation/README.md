# 07_Performance-Evaluation

This folder contains all documentation related to monitoring, measurement, analysis, evaluation, and auditing activities required for an effective Information Security Management System (ISMS) under ISO/IEC 27001:2022.

Performance evaluation ensures the ISMS is functioning as intended, meeting objectives, and producing measurable improvements.

---

## This section includes:

- ISMS monitoring and measurement  
- Internal audit planning and execution  
- Audit criteria and evidence collection  
- Key performance indicators (KPIs)  
- Management review inputs and outputs  
- Nonconformity tracking  
- ISMS performance dashboards and metrics  

---

## Included Templates

- ISMS Monitoring & Measurement Plan  
- Key Performance Indicators (KPI) Tracker  
- Internal Audit Program  
- Internal Audit Checklist (Clauses + Annex A)  
- Audit Report Template  
- Nonconformity & Corrective Action Log  
- Management Review Input Worksheet  

---

## ISO/IEC 27001:2022 References

- **Clause 9.2** – Internal audit  
- **Clause 9.3** – Management review
