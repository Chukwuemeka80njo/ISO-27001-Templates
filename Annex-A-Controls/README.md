# Annex-A-Controls

This folder contains templates, guidance, and documentation for all **93 Annex A controls** defined in ISO/IEC 27001:2022 and described in ISO/IEC 27002:2022.

Annex A provides the complete set of security controls required to treat information security risks identified during the risk assessment process.

Controls are grouped into four major themes according to ISO/IEC 27002:2022:

---

## **A.5 Organizational Controls (37 controls)**  
Policies, roles, responsibilities, governance structures, supplier management, access rules, classification, and operational security processes.

## **A.6 People Controls (8 controls)**  
Screening, training, awareness, disciplinary processes, and responsibilities of employees and contractors.

## **A.7 Physical Controls (14 controls)**  
Physical entry control, equipment security, environmental protection, asset storage, and secure facilities.

## **A.8 Technological Controls (34 controls)**  
Technical controls such as logging, monitoring, encryption, identity management, backup, configuration management, secure development, network security, and more.

---

## Included Templates

This folder will include:

- Control description and purpose  
- Control implementation guidance  
- Control evidence requirements  
- Maturity statements  
- Control monitoring & measurement  
- Example implementation procedures  
- Mapping to risk treatment decisions  
- Mapping to SoA (Statement of Applicability)  

Example templates provided:

- A.5 Organizational Controls Template  
- A.6 People Controls Template  
- A.7 Physical Controls Template  
- A.8 Technological Controls Template  
- Annex A Control Evidence Log  
- Annex A Implementation Checklist
